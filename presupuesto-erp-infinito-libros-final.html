Presupuesto: Sistema de Gestión para Infinito Libros

Propuesta por etapas para implementar un sistema propio de ventas, caja, stock, pedidos y automatizaciones operativas, adaptado a una librería de una sucursal y equipo reducido.

Propuesta comercial
Sistema web privado
1 sucursal · 3 vendedoras · 1 dueña

Enfoque de la propuesta

La propuesta se divide en un sistema base operativo y módulos adicionales. Esto permite comenzar con una versión útil para el trabajo diario y sumar automatizaciones según prioridad.

Importante: la demo presentada funciona como referencia visual. El sistema real requiere desarrollo con base de datos, usuarios reales, seguridad, pruebas, publicación y capacitación.

Implementación por etapas

No es necesario implementar todo desde el primer día. Se puede iniciar con el sistema base o recomendado, validar el uso real y luego sumar remitos, ARCA o WhatsApp según necesidad.

Sistema base operativo

Base

Ventas y comprobante interno

Registro de ventas, búsqueda de libros, carrito, descuentos, medios de pago y comprobante interno imprimible sin validez fiscal.

Base

Lector de códigos incluido

Preparación del sistema para usar pistola lectora de códigos de barras/ISBN en ventas y consulta de stock. El hardware no está incluido.

Base

Caja diaria

Apertura, ingresos por ventas, egresos, retiros, cierre, diferencias y movimientos asociados a cada usuaria.

Base

Stock por ISBN

Inventario de libros con título, autor, ISBN, editorial, categoría, precio, ubicación, stock actual y stock mínimo.

Base

Pedidos de clientes

Registro de pedidos con fecha, libro solicitado, link opcional, precio acordado, cliente, teléfono, observación y estado.

Base

Usuarios y reportes básicos

Roles simples: dueña y vendedoras. Reportes básicos de ventas, caja, stock bajo, ventas por vendedora y libros más vendidos.

Valor sistema base: USD 1.600

Adicionales cotizados por módulo

Módulo adicional Detalle Importe
Actualización masiva de precios por ISBN Carga de planilla del proveedor, validación de ISBN, comparación contra precio actual y actualización masiva de precios sugeridos. Incluye pantalla de revisión antes de aplicar cambios. USD 350
Consulta / carga asistida Top 10 Cúspide Consulta o carga asistida del ranking Top 10 para identificar libros con descuento especial. Si el sitio externo cambia o bloquea el acceso, se utilizará una alternativa manual/asistida. USD 250
Tutoriales e instructivos internos Apartado para subir, enlazar y clasificar videos de uso del sistema: ventas, caja, stock, pedidos, precios y situaciones cotidianas. USD 150
Carga asistida por remito PDF/imagen Subida de remito, detección asistida de libros y grilla de revisión para confirmar ISBN, cantidades, precios y ubicación antes de actualizar stock. Requiere revisión humana. USD 700
Facturación electrónica ARCA real Integración inicial para emisión de comprobantes desde el sistema. Alcance acotado a una empresa, una sucursal y un punto de venta, sujeto a relevamiento fiscal y pruebas de homologación. USD 900

Complemento opcional: WhatsApp con IA

Se puede incorporar una automatización para responder consultas frecuentes por WhatsApp y consultar datos del sistema, como stock, precio y ubicación de libros. Este complemento requiere que el sistema de gestión esté implementado, ya que necesita consultar la información del ERP.

Complemento Incluye Importe
WhatsApp + IA conectado al ERP Automatización con n8n e IA para recibir consultas, interpretar mensajes, consultar stock/precio en el ERP, responder al cliente y derivar a una persona cuando corresponda. Incluye flujo inicial de preguntas frecuentes y registro de consultas. USD 950
Aclaración: este complemento no se incluye dentro de los paquetes principales para no elevar la inversión inicial. Costos externos de WhatsApp/Meta, IA, hosting o servicios de terceros no están incluidos.

Incluye

  • Relevamiento inicial del flujo de trabajo de la librería.
  • Diseño funcional y visual de pantallas.
  • Desarrollo del sistema según el paquete contratado.
  • Roles simples: dueña y vendedoras.
  • Base de datos real para ventas, caja, stock y pedidos.
  • Publicación inicial del sistema web privado.
  • Capacitación inicial de uso.
  • Soporte correctivo durante 30 días posteriores a la entrega.
  • Hasta 2 rondas de ajustes menores dentro del alcance aprobado.
  • Documentación básica de uso para operaciones principales.

No incluye

  • Hardware: lector de códigos, impresora, PC, dominio, hosting o servicios externos.
  • Carga completa del catálogo histórico si no se entrega una planilla ordenada.
  • Cambios ilimitados o funcionalidades no detalladas en el paquete aprobado.
  • OCR automático sin revisión humana para remitos.
  • Costos externos de servicios de terceros.
  • Soporte mensual permanente luego del período correctivo inicial.
  • Configuración fiscal, clave fiscal o datos impositivos del cliente.
  • Operación multi-sucursal, multiempresa o contabilidad avanzada.
